## Features
- Home Page with a list of Categories
- Product Listing Page
- Filters 
- Individual Product Page
- Cart Management
- Wishlist Management
- Address Management
- Checkout
- Order Summary
- Payment Gateway Integration
- User Authentication
- Error Page
- Alerts and Loader
- Responsive

## Built Using:-
- ReactJS
- Vanilla CSS
- React Router v6
- React Context API + useReducer
- mockBee for backend
- Razorpay Payment Gateway
